# Android-unpack-tool
## 如何安装?
终端输入

```bash
git clone https://github.com/wwrrj/Android-unpack-tool.git
```
## 如何使用？
手机端安装好Linux Deploy

如何安装这里不再赘述

连接ssh，进入工具目录

终端输入

```bash
su
```

```bash
cd Android-unpack-tool
```

```bash
sudo bash menu.sh
```

即可使用
