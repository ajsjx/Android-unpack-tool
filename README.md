# Android-unpack-tool
## 如何使用？
手机端安装好Linux Deploy

如何安装这里不再赘述

链接ssh

终端输入

```bash
git clone https://github.com/wwrrj/Android-unpack-tool.git
```

```bash
cd Android-unpack-tool
```

```bash
sudo bash menu.sh
```

即可使用

## 首次使用请在菜单内安装依赖，否则无法使用
